Okta is a standards-compliant OAuth 2.0 authorization server and a certified OpenID Provider.

Okta's OAuth 2.0 API provides API security via scoped access tokens, and OpenID Connect provides user authentication and an SSO layer which is lighter and easier to use than SAML.

This site shows how Okta integrates with several leading API gateways. However, any gateway that supports an external OAuth authorization server should work with Okta. Similarly, any gateway that supports jwt validation should work with Okta as well.
